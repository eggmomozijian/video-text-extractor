# 📤 如何分享给朋友

## 方法一：打包整个项目（推荐）

### 1. 压缩项目文件夹

将整个 `shipintiquwenzi` 文件夹压缩成 ZIP 文件

**排除以下文件夹（可以减小体积）：**
- `node_modules` （朋友运行时会自动安装）
- `venv` （朋友运行时会自动创建）
- `uploads` 
- `downloads`
- `.git`

### 2. 发送给朋友

通过以下方式发送：
- 网盘（百度网盘、阿里云盘）
- 微信/QQ（如果文件不大）
- U盘

### 3. 朋友的操作步骤

1. 解压 ZIP 文件
2. 按照 `安装指南.txt` 安装必要软件
3. 双击 `启动应用.bat`
4. 首次运行会自动安装依赖（需要 5-10 分钟）
5. 浏览器自动打开，开始使用！

## 方法三：制作独立安装包（高级）

如果想做成真正的 EXE 安装包，可以使用：

### 前端打包：
```bash
npm run build
```

### 后端打包（使用 PyInstaller）：
```bash
pip install pyinstaller
pyinstaller --onefile --add-data "downloads;downloads" --add-data "uploads;uploads" app.py
```

但这种方式比较复杂，推荐**方法一**（打包文件夹）最简单！

---

## 📋 压缩前的检查清单

在压缩项目前，确保删除这些文件夹以减小体积：

- [ ] `node_modules/` （100MB+）
- [ ] `venv/` （500MB+）
- [ ] `uploads/` 
- [ ] `downloads/`
- [ ] `.git/` （如果不需要版本历史）

压缩后的文件应该在 **5-20MB** 左右
